사용 방법

✂️ 페이지 자르기

  1. PDF 파일 1개를 업로드
  2. 썸네일을 클릭해 페이지 선택 (체크 표시)
  3. 🔍 아이콘으로 페이지 확대 미리보기
  4. 🗑 아이콘으로 불필요한 페이지 삭제
  5. 드래그해서 페이지 순서 재배열
  6. 선택 페이지 추출 → 파일명 입력 → 다운로드

🔗 PDF 병합

  1. PDF 파일 2개 이상 업로드
  2. 목록에서 순서를 드래그해 조정
  3. PDF 합치기 버튼 클릭
  4. 파일명 입력 후 다운로드

🔒 개인정보 안내

  • 업로드된 PDF는 서버로 전송되지 않습니다
  • 모든 처리는 브라우저 내부에서 실행됩니다
  • 브라우저를 닫으면 데이터가 사라집니다

🌐 지원 환경

  • Chrome, Edge, Firefox, Safari
  • 인터넷 연결 (첫 로드 시 라이브러리 다운로드)
  • 대용량 PDF는 처리에 시간이 걸릴 수 있습니다
